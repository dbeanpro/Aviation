CPDLC

Controller Pilot Data Link Communications (CPDLC) is a technology that allows pilots and air traffic control to communicate using text-messages. It's a key part of the future air navigation system (FANS), which aims to improve aviation safety and reduce errors by providing a common language that's not affected by accents and/or language barriers.
